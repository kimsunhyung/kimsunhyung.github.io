0727 환경변수 설치

LINUX 환경 변수 설치

Created: 2022년 7월 27일 오전 11:44
Last Edited Time: 2022년 7월 27일 오후 4:32

vi 편집기 중요

  • 환경변수 설치하기

pwd 입력하여 위치 확인 후 복사

vi ~/.bashrc 입력하면 편집기가 켜짐

i를 입력하면 insert 가 생김

  • 저장하기
  • 작성 후 ESC누른 후

:wq! 를 입력

  • 시스템에 반영을 하기 위해

source ~/.bashrc 를 입력

  • 반영 후 가상 환경에서 나가지기 때문에 다시 가상 환경을 설정한 후

  • 위와 같은 코드를 입력하면 환경설정 완료

잘못 작성했을 시 저장하지 않고 나감
